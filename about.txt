OpenGL.GL - модуль используется для создания графики.
OpenGL.GLU - модуль считается устаревшим, используется для формирования целостных фигур (шаров, кубов и т.п.).
OpenGL.GLUT - модуль считается устаревшим, используется для формирования окон и обработки входных событий.

GLFW - актуальный модуль, используется для формирования окон.
GLEW - актуальный модуль, используется для обработки входных событий.
